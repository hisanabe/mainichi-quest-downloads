# studay-downloads

Public APK downloads for STUDAY（スタデイ）.

The Mainichi Quest releases already present in this repository are retired legacy artifacts. They are not renamed or reused as STUDAY packages.

The clean-break initial Android release is published as `jp.studay.app` version `1.0.0` (`versionCode` 1). It does not migrate or reuse the legacy app's account, data, or package identity.

- APK: https://github.com/hisanabe/studay-downloads/releases/latest/download/studay.apk
- Update manifest: https://raw.githubusercontent.com/hisanabe/studay-downloads/main/app-version.json

The build, package metadata, SHA-256, and signing certificate have been verified. Physical-device install and update E2E remain pending.
